# Analisis-Statistik-Kemiskinan-Jawa-Timur-2025
Project ini membahas analisis statistik data kemiskinan kabupaten/kota di Jawa Timur tahun 2025 menggunakan data dari Badan Pusat Statistik (BPS). Analisis dilakukan untuk mengetahui jumlah penduduk miskin, persentase kemiskinan, proporsi data, serta variasi tingkat kemiskinan antar daerah.
